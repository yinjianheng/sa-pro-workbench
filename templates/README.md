# 解决方案架构师模板库 / SA Template Library

> 本目录包含售前-方案-投标全链路的标准模板。所有模板均为填空式设计，填入客户信息即可使用。

## 模板清单

| 编号 | 模板名称 | 用途 | 阶段 |
|------|----------|------|------|
| T01 | 方案计划 Solution Plan | 客户背景、机会分析、方案摘要、风险缓解、资源计划 | 售前 |
| T02 | 框架方案 HLD | 12章标准结构：概述→需求分析→架构设计→实施计划→运维→成本 | 方案 |
| T03 | 蓝图/初步设计 LLD | 10章标准结构：详细组件行为、接口、配置 | 交付 |
| T04 | SOW 工作说明书 | 8章标准结构：范围→交付物→时间→验收→假设→变更→报价→条款 | 合同 |
| T05 | RFP/RFI 逐条响应模板 | 评分索引表 + 商务/技术条款逐条响应 | 投标 |
| T06 | 竞品分析报告 | 竞品定位→功能对比→优劣势→应对策略 | 售前 |
| T07 | 高管简报 PPT 结构 | 5-8页：问题→方案→价值→路线图→下一步 | 售前 |
| T08 | 述标 PPT 结构 | 15-20页：公司→理解→方案→案例→团队→商务 | 投标 |
| T09 | 架构决策记录 ADR | 标题→状态→上下文→决策→后果 | 方案 |
| T10 | 客户沟通预判与应答策略 | 可能问题→标准应答→风险提示 | 售前 |
| T11 | PoC 方案模板 | 成功标准→测试范围→验证目标→时间计划 | 售前 |
| T12 | 投标材料包清单 | 商务标→技术标→报价清单→资质文件 | 投标 |
